概要
Unityを用いて制作した、クリック操作のみで遊べるシンプルなアクションゲームです。
プレイヤーは飴を発射して敵を倒し、ドロップしたハートを動く籠に入れることでスコアを伸ばしていきます。
操作を極限まで簡単にすることで、幼少期の子どもやゲームに慣れていない人でも直感的に楽しめることを目指しました。
短時間で繰り返し遊べ、スコアを競える設計になっています。
主な機能
・クリック操作のみで完結するシンプルな操作設計
・敵の出現・撃破によるスコア加算
・障害物の生成・移動
・ハート取得によるスコア倍率・加点システム
・ダメージ処理
・ゲームオーバーなどの進行管理
・短時間で繰り返し遊べるゲームループ設計
